Blog post: Here is my coding routine!


You can use it for a bit of inspriation! But it isn't perfect for everone, make sure that you find your own way!

First Step - prepare the project

First, you have to find the right language for your project. As an example, it doesn't make sence to use python if you want to prepare an android app. So find the right language. Then think about your goal clearly and think about, who could use your project and why. Then you have to create a detailed projects plan including a timeline and a list of tasks. As the fourth step you have to instgall the right tools and (if needed ) learn a new language or framework.

Second step - coding

While coding, you have to write clear and structured code, name your variabels with good names. Then add comment to your code, so taht others can review your code. Another keypoint is, you have to find your design, it doesn't make sensce, that while you are programming, you are experiementing with the design of your project.

Third step - prepare relase

As the last keypoint, you have to write some tests, to find bugs and errors in your code, before you release your code on gIhtub maybe. Another point is that you have to write a documentation about your code, this will help the users to understand your project more clearly.

Thanks for reading !

I hope my coding routine can help you by your next project! Some last words: This routine isn't perfect for everyone, so find your own routine, but you can use mine for a bit of inspriation!
