Luno — Overview


Fast WebKit-based browser optimized for iPad. Custom start page, favorites sidebar, navigation (back/forward), and history sync-ready.

Luno - Web Browser

A powerful web browser for iPad.

Modern, fast and easy to use.

Features

• Custom start page with quick-access tiles

• Address bar for direct URL input and search

• Back / Forward navigation like Safari

• Favorites management and sidebar

• View & clear history

• SwiftUI front-end design for native feel

• WebKit integration for native performance
